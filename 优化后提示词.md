# Agent 面试题管理系统 - 开发提示词

## 角色设定

你是一位资深的全栈开发工程师与 AI 架构师，擅长 Agentic Workflow 开发。请根据以下需求完成开发设计与核心代码实现。

## 技术栈

- **前端**: Vue 3 (Composition API) + Vite + Tailwind CSS
- **后端**: FastAPI + Python 3.10+
- **LLM 框架**: LangChain
- **数据库**: SQLite (结构化存储) + ChromaDB (向量存储)
- **架构**: 前后端分离

## 功能模块

### 1. 题库管理

- **分类选择**: 单选按钮 (Agent / RAG / Memory / Tool Calling)
- **下一批**: 从 SQLite 随机查询 5 个已有题目
- **新问题**: 调用 LLM 实时生成 5 个该领域的面试题及答案
- **收藏**: 新问题显示"收藏"按钮，点击后存入 SQLite 并同步 Embedding 到 ChromaDB
- **已收藏问题**: 显示"生成新答案"和"删除"按钮
  - 删除: 从 SQLite 和 ChromaDB 同步删除
  - 生成新答案: 重新调用 LLM 生成答案并更新

### 2. 简历模拟面试

- **输入**: 用户粘贴简历中的项目描述
- **生成**: 提取关键技术点，生成 3 个针对性面试题及答案
- **下一批**: 再次生成 3 个问题 (深挖或扩展)
- **收藏/删除**: 同上

### 3. UI 设计

- 风格: 极简清爽，拒绝蓝紫色默认配置
- 色调: 灰白/冷色系，高对比度

## 工程化要求

2. 先完成后端模型和 LangChain Prompt，再开发前端
