## UI 风格说明

| 元素 | 设计 |
|------|------|
| 背景 | 浅灰 #F8FAFC |
| 卡片 | 白色 #FFFFFF，带轻微阴影 |
| 文字 | 深灰 #1E293B |
| 强调色 | 冷灰 #475569 或 墨绿 #059669 |
| 按钮 | 简洁边框样式，无渐变 |
| 间距 | 宽松，留白呼吸感 |
| 圆角 | 小圆角 4-6px |
